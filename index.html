				background-color: background;
				color: color;
				border: 2px solid border-color;
			}

			.btn-1 {

Реальность

			@mixin customBtn($background, $color, $border-color) {
				background-color: $background;
				color: $color;
				border: 2px solid $border-color;
			}

			.btn-1 {
				@include customBtn(#FFEB3B, #000, #000)
			}
			.btn-2 {
				@include customBtn(#F48FB1, #fff, #903075)
			}

Спецификация

Факты

  1. Кандидат в рекомендации с 3 декабря 2015 года
  2. Поддержка:
    Firefox - c 22 июля 2014
    Chrome, Safari - c марта 2016
    Edge - c 17 октября 2017
  3. Можно использовать как в стилевых файлах, так и в svg и html
  4. Значение может быть абсолютно любым

Преимущества

  1. CSS переменные живут в DOMе
  2. Работают по правилам каскада
  3. Могут изменяться из JS

Синтаксис

			:root {
				--font-size: 20px
			}
			body {
				font-size: var(--font-size)
			}

Работают в @media выражениях

			:root {
				--font-size: 20px
			}
			@media(max-width: 768px) {
				:root {
					--font-size: 16px
				}
			}
			body {
				font-size: var(--font-size)
			}

Математические выражения

			:root{
				--base-size: 10px;
				--size-xl: calc(2*var(--base-size));
				--size-l: calc(var(--base-size) + 2px);
				--size-s: calc(var(--base-size) - 2px);
				--size-xs: calc(var(--base-size) / 2);
			}

Шрифт в зависимости от языка

			var userLang = window.navigator.language || navigator.userLanguage;
			switch (userLang) {
				case 'en':
					root.style.setProperty('--font', 'url("../fonts/font-latin.woff2")');
					break;
				case 'ru':
					root.style.setProperty('--font', 'url("../fonts/font-cyrillic.woff2")');
					break;
				default:
					root.style.setProperty('--font', 'url("../fonts/font.woff2")');
			}
